## Shower Key Features
{:.slide}

1. Built on HTML, CSS and vanilla JavaScript
2. All modern browsers are supported
3. Slide themes are separated from engine
4. Fully keyboard accessible
5. Printable to PDF

Shower ['ʃəuə] noun. A person or thing that shows.
