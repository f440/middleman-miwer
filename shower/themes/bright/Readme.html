Bright theme for Shower

Bright theme for shower engine, part of template.

To see Bright theme in action:

  • Open shwr.me/shower/themes/bright/
  • Click any slide to enter presentation mode
  • Use arrow keys to navigate
  • Press Esc to exit presentation mode

Theme is included by default to shower and template, but you can also download theme as an archive: shwr.me/bright.zip.

See more details in Shower readme. Follow @shower_me for support and updates. Use Issues for sharing your ideas, bugs and feature requests.
